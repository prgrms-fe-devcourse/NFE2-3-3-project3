## 📝 Commit Convention

- ✨:sparkles: feat 기능 (새로운 기능)
- 🐛:bug: fix 버그 (버그 수정)
- ♻:recycle: refactor 리팩토링
- 💄:lipstick: style 스타일 (코드 형식, 세미 콜론 추가: 비즈니스 로직에 변경 없음), UI 수정
- 📝:memo: docs 문서 (문서 추가, 수정, 삭제)
- ✅:white_check_mark: test 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없음)
- 🔨:hammer: chore 기타 변경 사항 (빌드 스크립트 수정 등)
- 🎨:art: add 파일 추가, 보완, 코드 몇 줄 추가

## ✅ Implementation List

### 기본 요구사항

- [x] 웹 서비스 기본 프론트 제작
- [x] SNS를 활용한 소셜 로그인(회원가입/로그인/로그아웃)
- [x] 게시글 작성/수정/삭제
- [x] 게시글 이미지 첨부
- [x] 게시글 상세 조회
- [x] 타인의 게시글 반응, 댓글 작성/수정(블로그라면 타인의 블로그 글을 기준으로)
- [x] 엉뚱한 페이지에 접속하면 404 페이지를 보여주세요.

### 보너스 요구사항

- [x] 유저 팔로우/언팔로우/팔로잉 리스트 확인
- [x] 내 정보 확인
- [x] 회원 정보 수정
- [x] 다크 모드를 적용해보세요.

### 보너스 구현사항

- [x] AI를 이용한 분석, 그림 생성, ASMR 추천 가능
- [x] 반응형으로 다양한 디바이스에서 이용 가능
- [x] 카카오 공유하기 기능으로 간편하게 포스트를 공유
- [x] 다중 이미지를 업로드 가능
- [x] 경고 메세지 출력 시 커스텀 모달이 출력
- [x] 알림 모두 삭제, 모두 읽음, 하나씩 읽음 가능
- [x] 검색, 인기순 ,최신순 페이지 네이션 가능
